# Projet TodoList Sonny LALLIER

```shell
# Pour installer tous les packages
npm install
```

```shell
# Lancer en dev
npm run dev
```

```shell
# Build de prod
npm run build
```

```shell
# Lancer de serveur de preview ("prod")
npm run preview
```

## Tests

```shell
# Pour lancer les tests
npm run build && npm run preview

# Puis dans un autre terminal
cd tests && npm run open
```
